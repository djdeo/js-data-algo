# js-data-algo

## HashTable

```bash
const ht = new HashTable()
ht.put('abc', '123')
ht.put('bbc', '234')
ht.put('cbc', '345')

console.log(ht.get('abc'))
```

## DoublyLinkedList

```bash
const list = new DoublyLinkedList()

list.append('abc')
list.append('4dd')
list.append('tqsd')
```
