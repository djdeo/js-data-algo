# js-data-algo

## HashTable

```bash
const ht = new HashTable()
ht.put('abc', '123')
ht.put('bbc', '234')
ht.put('cbc', '345')

console.log(ht.get('abc'))
```

## DoublyLinkedList

```bash
const list = new DoublyLinkedList()

list.append('abc')
list.append('4dd')
list.append('tqsd')
```
## BinarySearchTree

```bash
const bst = new BinarySearchTree()

bst.insert(11)
bst.insert(7)
bst.insert(15)
bst.insert(3)
bst.insert(9)
bst.insert(8)
bst.insert(10)
bst.remove(7)
```
